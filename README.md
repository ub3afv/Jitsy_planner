# 1. Скачать скрипт
wget http://raw.githubusercontent.com/ub3afv/Jitsy_planner/refs/heads/main/install-full.sh

# 2. Дать права на выполнение
chmod +x install-full.sh

# 3. Запустить установку
sudo ./install-full.sh

2. Настройка OAuth2 в Nextcloud (для единого входа)
Откройте: https://cloud.my-domain.ru/settings/admin/security
Перейдите в раздел «Безопасность» → «OAuth 2.0»
Нажмите «Добавить клиент»
Заполните:
Имя клиента: Jitsi Meet Planner
Редирект URI: https://meet.my-domain.ru/api/auth/nextcloud/callback
Скопируйте полученные Client ID и Client Secret в .env
